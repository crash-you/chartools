通用问题

所有工具都是免费的吗？

是的，我们提供的所有文字处理工具都完全免费使用，无需注册或付费。我们致力于为用户提供高质量的免费服务，帮助大家提升工作效率。未来也将继续保持免费开放。

需要注册账号吗？

不需要。所有工具可以直接使用，无需注册、登录或提供任何个人信息。打开网页即可立即使用，这样既保护了您的隐私，又节省了您的时间。

支持哪些浏览器？

我们的工具支持所有现代浏览器，包括：

  • Chrome 90+ （推荐）
  • Firefox 88+
  • Safari 14+
  • Edge 90+

建议使用最新版本的浏览器以获得最佳体验。不建议使用IE浏览器。

移动设备可以使用吗？

完全可以！我们的网站采用响应式设计，完美适配手机和平板设备。在移动设备上，界面会自动调整为竖屏布局，操作体验流畅。支持iOS和Android系统的所有主流浏览器。

工具使用

工具有文本大小限制吗？

建议根据不同工具选择合适的文本大小：

  • 推荐：单次处理不超过100万字符（约50万汉字）
  • 可接受：100-200万字符（可能稍慢）
  • 不推荐：超过200万字符（可能卡顿）

对于特别大的文件，建议分段处理或使用专业的桌面软件。

如何保存处理结果？

每个工具都提供"复制"按钮，点击即可复制结果到剪贴板。然后您可以粘贴到任何地方使用（Word、记事本、编辑器等）。部分工具还支持浏览器截图或打印功能。

工具之间可以配合使用吗？

可以！我们的工具设计为可以组合使用。常用组合：

  • 数据清洗：文本转换（去重、排序）→ 字符计数（统计结果）
  • 代码审查：格式化工具（美化代码）→ 文本比较（查看差异）
  • 写作优化：字符计数（控制字数）→ 文本转换（格式调整）→ 文本比较（版本对比）

简繁体转换需要联网吗？

首次使用简繁体转换功能时需要联网，系统会从CDN加载OpenCC转换库（约200KB，通常3-5秒完成）。加载完成后，转换库会缓存在浏览器中，后续使用无需网络。其他所有功能（大小写、排序、格式化等）都无需联网即可使用。

隐私与安全

我的文本会被上传到服务器吗？

绝对不会！这是我们最重要的承诺。我们的工具采用纯前端技术，所有文本处理都在您的浏览器本地完成。您输入的任何文本都不会离开您的设备，不会上传到我们的服务器或任何第三方服务器。即使断开网络连接，工具依然可以正常使用（简繁转换需首次联网加载库）。

如何验证数据不会上传？

您可以通过以下方式验证：

  1. 打开浏览器的开发者工具（F12）
  2. 切换到Network（网络）标签
  3. 使用我们的工具处理文本
  4. 观察网络请求列表，您会发现没有任何数据上传请求
  5. 也可以断网后测试，工具依然正常工作

我们的代码也完全开源在GitHub，欢迎审查。

可以处理敏感信息吗？

可以安全处理。由于所有操作都在本地完成，您可以放心处理包含敏感信息的文本，如：个人身份信息、商业机密、代码、Token等。处理完成后关闭页面，不会留下任何痕迹。

网站使用Cookie吗？

我们仅使用必要的Cookie用于：Google Analytics匿名访问统计（帮助我们改进服务）和语言偏好设置。不使用Cookie追踪个人信息或用户行为。您可以在浏览器中禁用Cookie，工具功能不受影响。

技术问题

工具的性能如何？会卡顿吗？

我们的工具经过性能优化，大部分操作都能快速完成：

  • 字符计数：实时更新，100K字符 < 100ms
  • 大小写转换：< 50ms
  • JSON格式化：1MB文件 < 300ms
  • 文本对比：5000行 < 1秒

对于特别大的文本，可能会稍慢，但仍在可接受范围内。

为什么选择纯前端技术？

纯前端技术带来多重优势：

  • 隐私保护：数据不离开您的设备，100%安全
  • 快速响应：无需服务器往返，处理速度更快
  • 离线可用：无需网络也能使用（部分功能首次需要加载库）
  • 成本节约：无服务器成本，可以永久免费提供
  • 透明可审：代码开源，用户可以查看和验证

可以离线使用吗？

大部分功能支持离线使用。工具加载完成后，即使断网也能正常工作。唯一的例外是简繁体转换功能首次使用时需要联网加载OpenCC库（约200KB），加载后即可离线使用。

工具会自动保存我的输入吗？

不会。我们不会自动保存您的输入内容。关闭页面或刷新后，输入的文本会清空。如果您需要保存内容，请使用"复制"功能将结果复制到其他地方保存。这样设计是为了保护您的隐私。

如何快速访问工具？

建议的快速访问方法：

  • 浏览器书签：按Ctrl+D将常用工具加入书签
  • 设为主页：将工具页面设为浏览器主页
  • 移动端：将网站添加到手机主屏幕
  • 快速链接：从首页工具列表快速跳转
