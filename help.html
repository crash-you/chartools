开始使用

文字处理工具网站提供多种文本处理功能，让您轻松处理各种文字相关任务。所有工具都在浏览器中运行，无需安装任何软件。

如何开始？

  1. 打开首页，选择您需要的工具
  2. 根据工具界面提示输入文本
  3. 进行相应操作后查看结果
  4. 根据需要复制或下载处理后的文本

我们的所有工具都设计得简单易用，界面清晰直观，即使是第一次使用也能快速上手。

工具使用指南

字符计数工具

字符计数工具可以帮助您统计文本中的字符数、单词数、段落数等信息，并提供阅读时间估算。

使用步骤：

  1. 访问字符计数工具页面
  2. 在文本框中输入或粘贴您需要统计的文本
  3. 实时查看统计结果，包括总字符数、不计空格的字符数、单词数、段落数等
  4. 查看估算的阅读时间

提示：

字符计数是实时进行的，您无需点击任何按钮即可看到结果。如果您需要统计大量文本，可以先将文本粘贴进来，然后进行查看。

文本比较工具

文本比较工具可以帮助您对比两段文本，高亮显示差异，支持多种对比选项和文本合并功能。

使用步骤：

  1. 访问文本比较工具页面
  2. 在左侧文本框中输入或粘贴原始文本
  3. 在右侧文本框中输入或粘贴需要比较的文本
  4. 点击"比较文本"按钮开始对比
  5. 查看高亮显示的差异部分
  6. 可以根据需要调整比较选项，如忽略大小写、忽略空白字符等
  7. 如果需要，可以使用合并功能将两段文本整合

提示：

当比较较长文本时，系统会自动滚动到第一处差异位置。您可以使用"下一处差异"和"上一处差异"按钮在不同差异之间导航。

文本转换工具

支持大小写转换、简繁体转换、行处理、空白处理、全角/半角等常用文本转换操作，帮助您快速规范文本格式。

使用步骤：

  1. 打开 文本转换工具页面
  2. 在左侧输入框输入或粘贴文本
  3. 在右侧选择需要的转换操作（如转大写、转简体、升序排序等）
  4. 如需保存结果，点击“复制文本”复制处理后的内容

功能说明：

  • 大小写：转大写、转小写、标题化、大小写互换
  • 简繁体：转简体、转繁体（首次使用会自动加载转换库）
  • 行处理：升序/降序排序、去重、反转行、裁剪每行、移除空行
  • 空白处理：裁剪首尾、合并多空格、Tab→空格、移除多余空白行
  • 全角/半角：ASCII 字符全角与半角互转

提示：

简繁体转换依赖 opencc-js 库，首次点击相关按钮会从 CDN 自动加载。若网络不可达，仅简繁体转换受影响，其他转换功能不受影响。

格式化工具

支持 JSON、XML、HTML、CSS 的格式化与最小化，帮助你快速美化或压缩文本/代码。

  1. 打开 格式化工具页面
  2. 在左侧输入框粘贴文本，右侧选择类型与缩进
  3. 点击“格式化”或“最小化”查看结果，并可复制

提示：

JSON 将进行语法校验；XML/HTML/CSS 为轻量格式化器，适合通用场景。如需专业语法级格式化，可考虑在本地使用 IDE 插件。

常见问题解答 (FAQ)

这些工具是免费使用的吗？

是的，我们提供的所有工具均可免费使用，无需注册或登录。

我的文本数据安全吗？

所有文本处理都在您的浏览器中完成，数据不会传输到我们的服务器。您可以放心使用，不用担心数据泄露。

工具支持哪些语言的文字？

我们的工具支持多种语言，包括中文、英文等。对于字符计数工具，它可以处理几乎所有Unicode字符。

文本有大小限制吗？

由于所有处理都在浏览器中进行，过大的文本可能会影响性能。我们建议一次处理的文本不超过100万字符。对于特别大的文件，您可以分批处理。

我可以离线使用这些工具吗？

目前我们的工具需要在线使用。不过，我们正在考虑开发支持离线使用的应用版本，敬请期待。

简繁体转换是否需要联网？

是的，简繁体转换首次需要从 CDN 加载 opencc-js 库。加载完成后即可正常使用。若加载失败，您仍可使用大小写、行处理、全角半角等其他功能；也可联系管理员配置内网可访问的 CDN。

使用技巧

快捷键使用

大多数工具支持常见的键盘快捷键，例如：

  • Ctrl+A：全选文本
  • Ctrl+C：复制选中文本
  • Ctrl+V：粘贴文本
  • Ctrl+Z：撤销操作

高效使用字符计数工具

如果您需要统计多段文字，可以：

  • 使用分隔符（如空行）分隔不同段落，以获得更准确的段落计数
  • 利用实时统计功能，可以边编辑边查看字数变化
  • 对于需要控制字数的写作（如论文、简历等），可以边写边统计，确保不超过字数限制

浏览器兼容性

为获得最佳体验，我们建议使用最新版本的Chrome、Firefox、Edge或Safari浏览器。某些旧版浏览器可能无法正常显示或使用某些功能。
