🌟 开源贡献指南

贡献流程

1

Fork 项目仓库

访问GitHub仓库，点击右上角Fork按钮，创建你自己的副本

2

克隆到本地

使用git clone命令下载代码到本地

git clone https://github.com/your-username/toolsWords.git
3

创建分支

为你的功能或修复创建新分支

git checkout -b feature/your-feature-name
4

进行修改

修改代码、添加功能或修复bug，确保遵循代码规范

5

提交更改

提交你的修改并推送到GitHub

git add .
git commit -m "描述你的修改"
git push origin feature/your-feature-name
6

创建Pull Request

在GitHub上创建PR，详细描述你的更改和原因

✅ 贡献类型

  • • 修复bug和错误
  • • 添加新功能
  • • 改进UI/UX
  • • 优化性能
